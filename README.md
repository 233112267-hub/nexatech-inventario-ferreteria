# 📋 Descripción
GestiónFácil es una plataforma web desarrollada para optimizar la administración de inventario y ventas de una ferretería. El sistema permite controlar productos, registrar ventas, monitorear movimientos y generar reportes en tiempo real mediante una interfaz moderna y fácil de usar.

La problemática principal surge debido al uso de hojas de cálculo y procesos manuales, lo que provoca errores en inventario, pérdida de mercancía y dificultades para gestionar el stock y las ventas.

El sistema busca mejorar la organización operativa del negocio mediante herramientas digitales accesibles y escalables.
